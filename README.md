# appengine-nest-angular

Example full-stack Typescript project using:

* AppEngine Standard
* NodeJS 10.x
* NestJS 5.6.x
* Angular 7.2.x


## Installation

Install backend dependencies using:

    cd backend
    npm install

Install frontend dependencies using:

    cd frontend
    npm install


## Usage

Run the Nest backend:

    cd backend
    npm run start:dev

View the Nest backend at:

    http://localhost:8080/

Run the Angular frontend:

    cd frontend
    npm start

View the Angular frontend at:

    http://localhost:4200/


## Deployment

Deploy backend service:

    cd backend
    gcloud init
    npm install
    npm run deploy

Deploy frontend service:

    cd frontend
    gcloud init
    npm install
    npm run deploy


## Directory structure

    /backend                               --> Backend source files
    /frontend                              --> Frontend sources files
